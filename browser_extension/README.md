# AURA Browser Bridge Extension

Companion browser extension for AURA. Reads page content, tab information,
and dynamic updates, communicating with the AURA Python backend over a local
WebSocket connection.

## Installation (Development)

1. Open Chrome/Edge → `chrome://extensions`
2. Enable "Developer mode"
3. Click "Load unpacked" → select this `browser_extension/` directory
